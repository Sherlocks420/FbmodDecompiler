# DISCLAIMER:
The author does not condone or encourage the unauthorized reversing of mods into projects, and never has/never will steal other people's work. This tool was developed for educational purposes as a response to a recently popularized method (pre 2025) to rip mods by reading files from the Frosty ModData directory post mod application, which doesn't appear obfuscated anymore unlike .fbmods. In ApexGamemasters, we were personally affected by this circumstance, so in order to avoid individuals from abusing this loophole for continuous clout appropriation/scams/impersonation, the decision was made to bring awareness to such mod "reversing" phenomenon being possible by releasing a tool which makes it accessible for everyone.

## HOW TO USE:
Browse/drag and drop the .fbmod, pinpoint game's location and select .fbproject output folder.

Choose between .fbproject or .bin.

Press "reverse" when done

**Advanced:**
*EBX decompilation* will exclusively decompile EBX related assets such as blueprint/data containers.
*Chunk decompilation* will exclusively decompile textures, materials, sounds, models.

**Debugging:**
*Inspect* will provide any relevant information about any project.
*Compare* will provide insight in differences between two inputted projects.

**Diagnose:**
*Diagnose* will provide very detailed insightes of the selected .fbmod
